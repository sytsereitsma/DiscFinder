2. Introduction to MCU inside M100 chip
The M100 chip has a built-in 8-bit 8051 MCU, 256Byte internal memory and 16Kbyte program memory and 3 timers (Timer2 is used for
Baud rate generator, Timer0 is used for frequency hopping sequence control, Timer1 can be used by users). At the same time, built-in 8Kbyte data RAM,
The 8051MCU is shared with the digital demodulation circuit. When receiving data from the tag, the data RAM cannot be accessed by the MCU.
The MCU firmware can be downloaded from the external I2C EEPROM through the UART serial port or GPIO (P1.0 and P1.1) of the M100 chip.
The UART serial port data bits are 8 bits, 1 stop bit, and no parity bit.
